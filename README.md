# 🎵 Spain Top-50 Playlist Lifecycle Analysis

An end-to-end data analytics project that analyzes how songs perform over time in the Spain Top-50 playlist.  
The project focuses on understanding the **lifecycle of songs** — from entry to peak to exit — using data-driven techniques and an interactive dashboard.

---

## 🚀 Project Overview

With the rise of music streaming platforms, playlist rankings have become a key indicator of song success.  
This project transforms daily playlist data into meaningful insights by analyzing:

- How long songs stay in the playlist  
- How quickly songs reach peak position  
- Playlist stability and churn behavior  
- Performance differences based on song attributes  

The final output includes a **Streamlit dashboard** for interactive exploration and a **research paper** summarizing findings.

---

## 📊 Key Features

- 📈 Lifecycle Analysis of Songs  
- 📊 KPI Computation (Retention, Churn, Peak Time, Stability)  
- 🎯 Content Comparison (Explicit vs Clean, Single vs Album)  
- 📉 Playlist Churn & Stability Analysis  
- 🔍 Song Search & Exploration  
- 🌐 Interactive Dashboard (Streamlit)

## 📊 Key Insights

- Songs stay on the playlist for ~48 days on average  
- Peak position is usually reached within 9–10 days  
- Playlist shows low churn (~1–2 songs/day)  
- Singles outperform album tracks significantly  
- Clean songs have slightly better retention than explicit songs
